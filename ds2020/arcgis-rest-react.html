ArcGIS REST JS

Simplifies making requests to ArcGIS Online and Enterprise

For node & browser applications

Used in production (Hub, Storymaps, others)

But not an official product

Open source (PRs welcome :))

https://github.com/Esri/arcgis-rest-js

Handles:

  • Auth
  • Working with ArcGIS Online/Enterprise content and users
  • Querying and editing features in hosted feature layers
  • Administering hosted services
  • Geocoding
  • Routing

Features & Benefits:

  • Ala carte / lightweight (worst case 14.7kb)
  • Use with any/no framework
  • Hides API idiosyncrasies
  • Provides informative error codes
  • Parameter encoding
  • Documentation

Zen of Auth

  1. Open popup:
    https://www.arcgis.com/…/authorize​?client_id=ABC​&redirect_uri=yourapp.com/redir.html​
  2. User enters creds
  3. ArcGIS Online validates creds, client_id, & redirect_uri
  4. Returns a 302 redirect to the URI specified on redirect_uri with a hash containing a token and other info
  5. Popup loads that page
  6. The redirect page grabs the token and other info off the hash and passes it to the app
  7. App stashes token, etc in memory and maybe also in cookie or localStorage
