ArcGIS REST JS

Simplifies making requests to ArcGIS Online and Enterprise

For node & browser applications

Used in production (Hub, Storymaps, others)

But not an official product

Open source (PRs welcome :))

https://github.com/Esri/arcgis-rest-js

Handles:

  • Auth
  • Working with ArcGIS Online/Enterprise content and users
  • Querying and editing features in hosted feature layers
  • Administering hosted services
  • Geocoding
  • Routing

Features & Benefits:

  • Ala carte / lightweight (worst case 14.7kb)
  • Use with any/no framework
  • Hides API idiosyncrasies
  • Provides informative error codes
  • Parameter encoding
  • Documentation
